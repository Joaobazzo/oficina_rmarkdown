Introduo

Esse relatrio tem o objetivo de mostrar os dados socioeconmicos da cidade de Aracaju, utilizando a estrutura de Rmarkdown. Markdown uma estrutura simples de formatao de relatrios, permitindo exportar documentos em formatos HTML, PDF and MS Word. Para mais detalhes sobre Rmarkdown, consultar http://rmarkdown.rstudio.com

Esse manual, assim como os dados esto disponveis para download na pgina https://github.com/Joaobazzo/oficina_rmarkdown.

Um importante manual para a criao de documentos em Rmarkdown est disponvel em:

  • R Markdown Cookbook https://bookdown.org/yihui/rmarkdown-cookbook/

Os requisitos de softwares para rodar esse exemplo so:

  1. R https://cran.r-project.org/bin/windows/base/

  2. RStudio Desktop Site: https://www.rstudio.com/products/rstudio/download/#download

  3. Latex Site: https://www.latex-project.org/get/#tex-distributions distribuio TexLive ou MikTek

Aps a instalao dos softwares, instalar pacotes no R. Rodar no console do RStudio os seguintes comandos:

install.packages("ggplot2")
install.packages("mapview")
install.packages("geobr")
install.packages("bookdown")
install.packages("knitr")
install.packages("rmarkdown")

Aps a instalao, podemos incorporar os pacotes na memria do R.

library(ggplot2)
library(mapview)
library(geobr)
library(bookdown)
library(knitr)
library(rmarkdown)

A configurao inicial do documento em Rmarkdown dever ser do seguinte modo:

---
title: "Relatrio do municpio"
author: "Seu nome"
date: "`r Sys.Date()`"
site: "https://github.com/Joaobazzo/oficina_rmarkdown"
output: 
  bookdown::html_document2: default
editor_options: 
  markdown: 
    wrap: 72
---

Existem 46 municpios na RGINT Aracaju. Uma visualizao razovel desse fenmeno seria atravs de mapas. No entanto, vale lembrar que o arquivo de geometria geom_muni no conta com informao de PIB per capita. Assim, vamos ter que adicionar essa funo no arquivo geom_muni.

# filtro da geometria
my_cities_geom <- geom_muni[geom_muni$code_muni %in% pib_my_rgint$code_muni,]

# adicionar atravs da funo merge
my_cities_geom <- merge(x = my_cities_geom
                        ,y = pib_my_rgint[,c("code_muni","PIB_capita")]
                        ,by = "code_muni")

Agora, podemos visualizar os valores de PIB per capita atravs da funo mapview::mapview() ou por plot usando o pacote ggplot2.

mapview::mapview(my_cities_geom["PIB_capita"])

ggplot(data = my_cities_geom)+
  geom_sf(mapping = aes(fill = PIB_capita))

Podemos deixar o grfico visualmente mais agradvel adicionando algumas configuraes na Figura @ref(fig:pib-v2).

ggplot(data = my_cities_geom)+

Situao do domiclio

Nesta ltima anlise, ns vamos fazer uma anlise exploratria da relao da evoluo da urbanizao de todos os municpios do Estado (no qual nossa cidade exemplo faz parte). A listagem mostra a preparao dos dados, enquanto a Figura @ref(fig:situacaodomi) ilustra a visualizao.

# verifica qual estado pertence a cidade
abbrev_state <- idh[idh$code_muni == 2800308,"abbrev_state"]
abbrev_state

## [1] "SE"

# pega dados de populacao do estado
pop_city <- pop[pop$abbrev_state == abbrev_state,]
# seleciona apenas a geometria do estado
geom_city <- geom_muni[geom_muni$abbrev_state == abbrev_state,]

# mescla os dados de geometria com populacao
# usando como link a coluna "code_muni"
muni_merge <- merge(x = geom_city[,c("code_muni","geom")]
                  , y = pop_city[,c("code_muni","abbrev_state","ano","situacao_do_domicilio","proporcao")] 
                  ,by = "code_muni")

# plot mapa usando ggplot2

ggplot(data = muni_merge)+
  geom_sf(mapping = aes(fill = as.numeric(proporcao)))+
  geom_sf(data = muni_merge[muni_merge$code_muni == "2800308",]
          ,color = "red")+
  # faceta
  facet_grid(rows = vars(ano),cols = vars(situacao_do_domicilio))+
  # cores
  scale_fill_continuous(type = "viridis")+
  # legendas
  labs(title = "Percentual da populao por situao de domiclio (%)",
       subtitle = "Mapa do estado",
       fill = "Proporo (%)")+
  # tema
  theme_void()
